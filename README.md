# 프로젝트 소개

유니티 3D를 기반으로 제작한 디펜스 게임입니다. 게임의 목표는 스킬을 조합하여 덱을 만들고 보드판에 스킬을 소환하고 머지하면서 몰려오는 적들을 처치해야 합니다. 플레이를 통해 골드와 열쇠를 획득하여 새로운 스킬과 장비를 획득하고 강화하면서 성장할 수 있습니다.
